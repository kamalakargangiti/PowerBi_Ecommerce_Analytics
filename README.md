# PowerBi_Ecommerce_Analytics

This project is an end-to-end Power BI analytics solution developed for a US-based e-commerce sales business. It includes data cleaning, shaping, advanced data modeling, DAX calculations, KPI design, and fully interactive dashboard creation. The goal of this project is to provide a real-time, dynamic, and insightful dashboard that helps business teams understand sales performance, customer behaviour, product trends, and profitability.

This project reflects my skills in:
- Power BI  
- Power Query  
- DAX  
- Data Modeling (Star Schema)  
- Analytical Storytelling  
- Dashboard UI/UX Best Practices
